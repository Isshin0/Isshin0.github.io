1 前置环境 （Node.js 、Git )

1. 1 Node.js

  • node下载官网

关于Node.js环境配置的详细：node.js安装及环境配置之Windows篇 by刘奇云

1.2 Git

  • Git下载官网
  • Windows系统Git安装教程（详解Git安装过程）by学有所用
  • Windows系统Git客户端安装教程2022版 by cy谭
  • 初次使用时需要进行 git账户验证，使用的是github提供的密钥进行绑定。

2 hexo

  • Hexo 官方网站
  • 具体细节查看文档即可。

2.1 npm install hexo-cli -g 安装

  • 指定文件夹Blog

2.2 hexo init 初始化

  • Create a new Hexo folder

2.3 hexo generate 生成页面

  • Generate static files.

2.4 hexo new 新建文章

  • Create a new post.

2.5 hexo server 启动本地服务

  • Start the server.

2.6 修改中文主题

  • ~/Blog/_config.yml

2.7 部署到github

1. 根目录配置文件修改

  • ~/Blog/_config.yml

2. github仓库新建

  1. github主页右上角的加号，New repository ；
  2. Repository name 命名遵循 yourname.github.io，yourname即为Owner名
